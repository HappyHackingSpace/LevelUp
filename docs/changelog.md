# Changelog

All notable changes to LevelUP will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- FastAPI backend implementation
- Database integration with SQLAlchemy
- Comprehensive API documentation
- Interactive Swagger/ReDoc docs
- Docker support

### Changed

- Refactored from single Streamlit app to API-based architecture
- Improved configuration management
- Enhanced error handling

### Fixed

- Database initialization issues
- Import path configuration
- Alembic path settings

## [0.1.0] - 2024-12-XX

### Added

- Initial release
- Streamlit web interface
- Gemini AI integration
- Multi-language support (EN, DE, FR, IT, RU, TR, ES)
- CV analysis features:
  - Career domain matching
  - Competency evaluation
  - Strategic insights
  - Development recommendations
  - Comparative benchmarking
- PDF text extraction
- Environment configuration with .env
- Comprehensive README
